# 지식 저장소 분류 체계

## 여신
- 여신/총칙: 여신업무 목적, 정의, 종류, 기본 원칙
- 여신/신용평가: 신용등급 체계, 평가 항목, 등급 조정 기준
- 여신/담보관리: 담보 종류, 가치 평가, 재평가 주기, 담보 보전
- 여신/심사절차: 신청 서류, 심사 단계, 승인·거절 기준, 실행
- 여신/사후관리: 정기점검, 연체관리, 워크아웃, 부실채권 처리

## 수신
- 수신/예금상품: 정기예금, 적금, 요구불예금, MMDA
- 수신/금리운용: 수신금리 결정 기준, 우대금리
- 수신/예금보호: 예금자보호법, 보호 한도 및 대상

## 외환
- 외환/환전: 환율 적용, 환전 한도, 절차
- 외환/해외송금: 송금 한도, 수수료, 신고 의무
- 외환/외화예금: 외화 보통예금, 외화 정기예금

## 리스크관리
- 리스크관리/신용리스크: PD·LGD·EAD 측정, 한도 관리, 충당금
- 리스크관리/시장리스크: 금리리스크, 환율리스크, VaR
- 리스크관리/운영리스크: 내부통제, 사고 예방, BCP

## 컴플라이언스
- 컴플라이언스/내부통제: 3선 방어, 준법감시, 이해충돌 방지
- 컴플라이언스/금융소비자보호: 적합성 원칙, 설명의무, 불완전판매
- 컴플라이언스/AML: 고객확인(CDD/EDD), 의심거래보고(STR), 고액현금거래(CTR)

## IT/보안
- IT/코어뱅킹: 계정계 시스템 운영 규정
- IT/정보보안: 접근통제, 패스워드 정책, 망분리
- IT/개인정보보호: 개인정보 처리, 유출 대응, 암호화
- IT/침해사고대응: 보안사고 탐지, 대응 절차, 당국 보고
