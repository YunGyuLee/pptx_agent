# 여신업무 규정 제2편 신용평가 및 등급체계

## 제7조 (신용평가 대상 및 주기)
1. 여신 신규취급 시 반드시 신용평가를 실시한다.
2. 기존 여신에 대한 정기평가는 연 1회 이상 실시한다.
3. 차주의 재무상태 급변, 부도, 소송 등 중대 사건 발생 시 수시평가를 실시한다.
4. 여신금액 50억 원 이상인 경우 반기 1회 이상 평가를 원칙으로 한다.

## 제8조 (기업 신용등급 체계)

### 8-1. 등급 분류
| 등급 | 분류 | 정의 |
|------|------|------|
| AAA | 정상 | 채무상환능력이 최상위 수준 |
| AA | 정상 | 채무상환능력이 매우 우수 |
| A | 정상 | 채무상환능력이 우수 |
| BBB | 정상 | 채무상환능력이 양호 |
| BB | 정상 | 채무상환능력이 보통이나 경기변동에 취약 |
| B | 요주의 | 채무상환능력이 다소 부족 |
| CCC | 요주의 | 채무상환능력이 불량하며 불확실성 높음 |
| CC | 고정 | 채무상환이 심각하게 위협받는 상태 |
| C | 회수의문 | 채무상환 불가능 가능성 매우 높음 |
| D | 추정손실 | 이미 부도 또는 채무불이행 상태 |

### 8-2. 평가 항목 및 배점
**재무평가 (60점)**
- 수익성: 영업이익률, ROA, ROE (15점)
- 안정성: 부채비율, 유동비율, 차입금의존도 (20점)
- 성장성: 매출성장률, 자산성장률 (10점)
- 활동성: 총자산회전율, 재고자산회전율 (15점)

**비재무평가 (40점)**
- 산업위험: 산업전망, 경쟁강도, 진입장벽 (15점)
- 영업위험: 시장지위, 제품다각화, 고객집중도 (15점)
- 경영위험: 경영진 역량, 지배구조, 내부통제 (10점)

## 제9조 (개인 신용평가)

### 9-1. 개인신용점수 활용
KCB 및 NICE 신용점수를 기본 평가 지표로 활용한다.
- 900점 이상: 1등급 (최우량)
- 800~899점: 2등급 (우량)
- 700~799점: 3등급 (양호)
- 600~699점: 4등급 (보통)
- 500~599점: 5등급 (주의)
- 500점 미만: 6등급 (불량, 원칙적 여신 제한)

### 9-2. 추가 평가 항목
- 소득 안정성: 고용형태, 근속연수, 소득 수준
- 부채 현황: 기존 대출 잔액, 연체이력, DSR
- 자산 현황: 보유 부동산, 금융자산, 보험

## 제10조 (신용등급 조정)
1. 재무평가와 비재무평가 결과를 합산하여 초기등급을 산출한다.
2. 다음 사유 발생 시 최대 2등급 하향 조정한다.
   - 최근 1년 내 30일 이상 연체 이력
   - 세금 체납 또는 강제집행 이력
   - 소송가액이 자기자본의 20% 초과
   - 주요 임원 횡령·배임 혐의 발생
3. 다음 사유 발생 시 최대 1등급 상향 조정 가능하다.
   - 우량 모기업의 실질적 지원 확약
   - 정부 또는 공공기관 보증 보유
   - 예금 등 적격 금융담보 제공

## 제11조 (부실징후 모니터링)
여신부는 다음 지표를 월별로 모니터링하여 부실징후를 조기 감지한다.
1. 당좌거래 정지 또는 어음 부도
2. 세금 체납 통보
3. 경영진 교체 또는 대주주 변동
4. 매출 전년 대비 30% 이상 감소
5. 주거래은행 변경
6. 주요 계약 해지 또는 소송 제기
