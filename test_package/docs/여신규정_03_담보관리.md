# 여신업무 규정 제3편 담보 및 보증 관리

## 제12조 (담보의 종류)

### 12-1. 물적담보
**부동산담보**
- 토지, 건물, 아파트, 상가, 공장 등 부동산에 저당권 설정
- 감정평가: 공인 감정평가법인 2개사 평균 또는 은행 자체평가
- 담보인정비율(LTV): 최대 80% (용도·지역에 따라 차등)
- 감정가 50억 원 초과 시 외부 감정평가 필수

**동산담보**
- 기계, 장비, 재고자산 등에 동산저당 또는 양도담보 설정
- 담보인정비율: 감정가의 50% 이내
- 6개월 이상 재평가 실시

**유가증권담보**
- 상장주식: 시가의 60% 이내 (최근 3개월 평균 종가 기준)
- 국공채: 액면가의 95% 이내
- 회사채(A등급 이상): 시가의 80% 이내

**예금담보**
- 정기예금, 정기적금 등 은행 수신 상품
- 담보인정비율: 원금의 95% 이내
- 담보설정 즉시 지급정지 처리

### 12-2. 인적담보 (보증)
**연대보증**
- 법인 대표자의 연대보증: 중소기업 여신 시 원칙적 징구
- 제3자 보증: 보증인의 재력 심사 필수 (여신금액의 150% 이상 순자산)

**기관보증**
- 신용보증기금, 기술보증기금, 지역신용보증재단
- 보증서 확인 후 대출 실행 (유효기간 반드시 확인)
- 보증비율: 보증서 기재 내용에 따름

## 제13조 (담보가치 평가기준)

### 13-1. 토지·건물 감정평가
1. 공시지가의 150% 이내 원칙 (서울·수도권 주거지역 제외)
2. 개별공시지가가 없는 경우 인근 유사지 비교 평가
3. 환경오염 토지, 분묘기지권 설정 토지는 감가 적용

### 13-2. 공장 및 특수부동산
1. 공장: 토지 + 건물 + 기계장치 일괄 평가
2. 골프장, 호텔 등 수익형 부동산: 수익환원법 병행 적용
3. 농지·임야: 개발가능성 제한 반영하여 보수적 평가

### 13-3. 담보 재평가 주기
| 담보 종류 | 재평가 주기 |
|-----------|------------|
| 부동산 | 2년 (여신금액 100억 이상: 1년) |
| 동산 | 6개월 |
| 상장주식 | 월별 시가 확인 |
| 예금 | 만기 도래 시 |

## 제14조 (담보 관리 및 보전)
1. 담보 설정 후 등기·등록 원부를 반기별로 확인한다.
2. 선순위 채권 변동 시 즉시 담보가치를 재산정한다.
3. 담보가치가 여신잔액의 110% 미만으로 하락 시 추가담보를 요구한다.
4. 화재보험 등 손해보험 가입 여부를 연 1회 이상 확인한다.
5. 경매·공매 진행 중인 담보에 대해 권리보전 조치를 즉시 실시한다.

## 제15조 (무담보 여신)
1. 신용등급 BBB 이상 법인에 한해 무담보 여신을 허용한다.
2. 무담보 여신 한도는 자기자본의 10% 이내로 제한한다.
3. 개인 신용대출은 DSR 40% 이내에서 취급한다.
4. 무담보 여신은 반기별 사후관리를 강화한다.
